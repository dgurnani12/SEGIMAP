RFC3501 - 7.4.2 - P.76-77

Returns a parenthesized list that described the envelope structure of a message. Computed by parsing the [RFC-2822] header into the component parts, defaulting various fields as necessary.

Requires (in the following order): date, subject, from, sender, reply-to, to, cc, bcc, in-reply-to, and message-id. The date, subject, in-reply-to, and message-id fields are strings. The from, sender, reply-to, to, cc, and bcc fields are parenthesized lists of address structures.
